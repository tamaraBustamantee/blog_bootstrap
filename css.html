Introducción a CSS

CSS Cascading Style Sheets es el lenguaje utilizado para describir la presentación y el diseño de las páginas web. Se utiliza para aplicar estilos a los elementos HTML, como colores, fuentes, márgenes, y más. CSS permite separar el contenido de la presentación, facilitando el mantenimiento y la actualización de los sitios web.
