Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear y diseñar páginas web. Se utiliza para describir la estructura de las páginas web mediante el uso de elementos y etiquetas. HTML es un lenguaje fundamental para el desarrollo web y es utilizado junto con CSS y JavaScript para crear sitios web interactivos y dinámicos.
